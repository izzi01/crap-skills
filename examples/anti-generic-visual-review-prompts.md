# Anti-Generic Visual Review Example Prompts

Use `skills/anti-generic-visual-review/SKILL.md` when the main issue is visual sameness, dribbblified polish, fake premium treatment, and weak ownability.

- "Review this landing page mockup: glow everywhere, mesh gradients, glass cards, stock faces, and generic startup polish. Tell me exactly why it feels interchangeable."
- "These three homepage variants all use the same hero rhythm, same card system, and same polished imagery. Critique the visual sameness and tell me how to create distinct silhouettes."
- "The founder says the page looks premium. I think it looks like premium theater. Review the visual direction harshly and suggest structural fixes."
