# Copy Checklist

## Audience
- Does the page name the real audience explicitly?
- Are secondary stakeholders named if they matter?

## Problem
- Does it name a real stuck moment, fear, or confusion?

## Mechanism
- Can the user tell how the offer works?
- Are the next steps visible?

## Proof
- Are there real proof surfaces?
- Are claims backed by evidence?

## Language quality
- Remove filler phrases.
- Remove internal/meta language.
- Keep route-specific voice intact.

## CTA
- Is the CTA concrete?
- Does it match actual user behavior?
