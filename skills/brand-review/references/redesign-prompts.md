# Redesign Prompt Starters

- "What makes this feel like trend fluency instead of brand identity?"
- "If the logo disappeared, what would still feel unmistakably ours?"
- "Where is the page using polish to hide weak hierarchy or weak proof?"
- "What one typographic or compositional move would make this more ownable?"
- "What would you remove first to stop this feeling AI-generated?"
- "Which line of copy is generic, and which visual treatment makes that genericness worse?"
- "If we stripped away gradients and glow, what would still carry trust and personality?"
- "Could this hero belong to ten other startups? If yes, why?"
