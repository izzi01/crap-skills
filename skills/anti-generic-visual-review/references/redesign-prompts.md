# Redesign Prompt Starters

Use these prompts to push beyond generic visual polish.

- "Make this page feel chosen, not generated. What should be removed first?"
- "If we removed all glow, blur, and gradients, what composition problems would remain?"
- "What would make this recognizably ours if the logo disappeared?"
- "Which image feels true, and which one only feels expensive?"
- "Where is the page using decoration to hide hierarchy problems?"
- "What one typographic or layout move could make this less template-like?"
- "If this were printed in black and white, what would still have personality?"
- "What parts look like Dribbble bait rather than a page built to help someone decide?"
