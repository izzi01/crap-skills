# Visual Style Guide Copy

This skill writes the text and document structure for visual style guides and brand-guideline artifacts.

It is based on external baseline sources from:
- `autumnsgrove/claudeskills@brand-guidelines`
- `adhi-jp/agent-skills@writing-style-guide`

This packaged version combines the strong document-template coverage from the brand-guideline source with the cleaner prose discipline from the writing-style-guide source.

## What this skill is for

Use it when the user wants to:
- create a visual style guide
- rewrite a brand-guidelines document
- document logo, color, typography, imagery, and layout rules
- write voice/tone and application guidance for a brand when it belongs inside a broader visual guide
- turn loose brand decisions into a durable guide others can use

## Core rule

A style guide is not decorative brand theater.

It should help a real reader:
- understand the brand system
- apply it consistently
- know what is allowed
- know what to avoid
- know where judgment is still required

## What strong style-guide copy does

It is:
- clear
- structured
- reference-friendly
- specific enough to apply
- short enough to use
- strict where consistency matters
- practical where teams need examples

## Workflow

### 1. Identify the guide's audience
Decide who will use it:
- designers
- marketers
- content creators
- agencies / vendors
- internal teams

The document should explain things at the level those readers need.

### 2. Capture the visual system
Gather or infer:
- brand story or positioning context
- personality traits
- logo system
- color palette
- typography system
- imagery or illustration style
- spacing / layout rules
- brand voice / tone rules
- common applications
- do / don't examples

### 3. Decide scope
The user may need:
- a full visual style guide
- a lightweight brand cheat sheet
- a single section rewrite
- voice/tone guidance embedded inside a visual guide

Do not overbuild the document if they only need one section.

### 4. Write for use, not admiration
Avoid fluffy brand language.

Prefer:
- direct section headers
- clear rules
- practical examples
- constraints people can follow
- notes on where exceptions are acceptable

## Writing principles

### Structure beats flourish
This is a reference artifact. Readers should be able to scan it.

### Rules need rationale
If a rule matters, explain why briefly.

### Examples prevent drift
Whenever possible, include:
- correct use
- incorrect use
- edge-case guidance

### Visual guidance needs text discipline too
Do not let the guide itself become marketing copy.

## Common sections

Use the sections that fit the request.

- purpose / how to use this guide
- brand story / mission / values
- brand personality
- target audience
- logo usage
- color palette
- typography
- imagery / photography / illustration
- layout / spacing / composition
- iconography
- brand voice & tone
- applications
- do / don't rules
- asset locations / contacts
